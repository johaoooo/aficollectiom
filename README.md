# AFI Collection — Site Web Officiel
> "L'art béninois, tissé à la main" — Fondée en 2015 par Mme TOSSA Afiavi Gbèssito Honorine

## Lancer en développement
```bash
# Frontend
cd front && npm install && npm run dev

# Backend (autre terminal)
cd back && npm install && npm run dev
```

## Stack
- Frontend : React 18 + Vite + Tailwind CSS
- Backend  : Node.js + Express
- Icônes   : Phosphor Icons
- Fonts    : Playfair Display + DM Sans
